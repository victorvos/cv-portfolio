# Agent rules

**Canonical rules** for this project live in **`.cursor/rules/`** (merged, generic structure).

- `.cursor/rules/` — Cursor IDE rules (general, operational, python, app_architecture, frontend, markdown, langchain_python).
- `.agent/workflows/` — Reference workflows (synced to `.cursor/workflows/`).
- `.agent/skills/` — Optional agent skills (architect, planner, etc.).

Use `.cursor/rules/` when editing; `.agent/rules/` here are legacy or for non-Cursor tooling.
